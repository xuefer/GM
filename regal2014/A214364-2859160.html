电路/系统说明
当要求发射器验证时，前后控制台和行李厢天线将同时在乘客舱内向发射器发射校验口令。如果发射器通过收音机频率 (RF) 的响应有效，则车身控制模块 (BCM) 将确定车内有发射器。如果未接收到响应，车身控制模块将确定车内没有发射器，驾驶员信息中心将显示“NO REMOTE DETECTED（未检测到遥控钥匙）”。
诊断帮助
发射器蓄电池电量过低或来自售后加装设备（如双声道收音机、电源逆变器、车载电话充电器、计算机等）的无线电频率 (RF) 的干扰，可导致系统故障。在无线电频率通信量较大的区域，例如使用无线电频率收发器刷卡付费的加油站，产生的干扰也可导致故障。
外部影响导致故障，这是正常的。按照本诊断不能诊断无线电频率干扰故障。如果无线电频率干扰可疑，则向客户询问发生“NO REMOTE DETECTED（未检测到遥控钥匙）”故障的情况或位置。

电路/系统检验
注意:该情况可能是由车辆蓄电池电压过低引起的。如果蓄电池没有充分充电（高于10伏），系统在检测发射器时会比较困难，并将显示“NO REMOTE DETECTED（未检测到遥控钥匙）”消息。校正蓄电池电压过低状况以排除该故障。
1.确认未设置DTC B3119、B3120、B3121、B3122、B3123或B3124。
如果已设置故障诊断码
参见故障诊断码 (DTC) 列表 - 车辆。
如果未设置任何故障诊断码
2.使用无钥匙进入发射器锁止车辆。
3.通过使用有效发射器接近车辆并操作相应车门外把手按钮，确认每个车门解锁并打开。
如果两个车门未解锁并打开
参见无钥匙进入系统故障（主动式）无钥匙进入系统故障（被动式）。
如果两个车门解锁并打开
4.确认发射器位于钥匙凹槽/槽中时车辆起动。
如果车辆不起动
执行带集成发射器的钥匙编程（带BTM）带集成发射器的钥匙编程（不带BTM）程序。
如果车辆起动
5.尝试起动车辆，并在每次尝试起动时将无钥匙进入发射器置于车内各个位置的同时，确认驾驶员信息中心不显示“NO REMOTE DETECTED（未检测到遥控钥匙）”信息。车辆应起动。
如果在任何时间显示“NO REMOTE DETECTED（未检测到遥控钥匙）”信息
参见“电路/系统测试”。
如果不显示“NO REMOTE DETECTED（未检测到遥控钥匙）”信息，且车辆起动
6.检查是否有任何干扰发射器和天线通信的无线电频率干扰源，例如车载电话充电器、电源等。如果未检查出导致无线电频率干扰的物件，则同客户讨论“NO REMOTE DETECTED（未检测到遥控钥匙）”状况出现的条件或位置。讨论车内正常保留的所有装置，例如送来维修时可能被拆下的笔记本或车载电话充电器。需要向客户解释无线电频率干扰如何降低系统的性能。
电路/系统测试
注意:要确认需断开并诊断的相应T10无钥匙进入天线，将发射器置于车内的各个位置并尝试起动。“NO REMOTE DETECTED（未检测到遥控钥匙）”消息出现时，发射器的位置将与需要诊断的T10无钥匙进入天线相符。
1.将点火开关置于“OFF（关闭）”位置，断开相应T10无钥匙进入天线处的线束连接器，将点火开关置于“ON（打开）”位置。
2.测试信号电路端子1和搭铁之间的电压是否低于1伏。
如果为1伏或更高
2.1 将点火开关置于“OFF（关闭）”位置，断开K84无钥匙进入控制模块处的线束连接器，将点火开关置于“ON（打开）”位置。
2.2 测试信号电路和搭铁之间的电压是否低于1伏。
如果为1伏或更高，则修理电路对电压短路故障。
如果低于1伏，则更换K84无钥匙进入控制模块。
如果低于1伏
注意:当使用“MIN MAX（最小最大）”功能时，数字式万用表上的自动量程可能不会正常工作。为了确保精确测试结果，应关闭自动量程功能并手动将数字式万用表的量程设置为000.0伏。
3.将点火开关置于“OFF（关闭）”位置，在信号电路端子1和搭铁之间安装一个数字式万用表。设置数字式万用表利用“MIN MAX（最小最大）”功能捕捉直流电压。
4.按下点火开关模式开关启动天线。
5.确认数字式万用表捕捉的最大电压高于1伏。
如果为1伏或更低
5.1 将点火开关置于“OFF（关闭）”位置，断开K84无钥匙进入控制模块处的线束连接器。
5.2 测试信号电路和搭铁之间的电阻是否为无穷大。
如果电阻不为无穷大，则修理电路对搭铁短路故障。
如果电阻为无穷大
5.3 测试信号电路端对端的电阻是否小于2欧。
如果为2欧或更大，则修理电路中的开路/电阻过大故障。
如果小于2欧，则更换K84无钥匙进入控制模块。
如果高于1伏
6.测试低电平参考电压电路端子2和搭铁之间的电压是否低于1伏。
如果为1伏或更高
6.1 将点火开关置于“OFF（关闭）”位置，断开K84无钥匙进入控制模块处的线束连接器，将点火开关置于“ON（打开）”位置。
6.2 测试低电平参考电压电路和搭铁之间的电压是否低于1伏。
如果为1伏或更高，则修理电路对电压短路故障。
如果低于1伏，则更换K84无钥匙进入控制模块。
如果低于1伏
注意:当使用“MIN MAX（最小最大）”功能时，数字式万用表上的自动量程可能不会正常工作。为了确保精确测试结果，应关闭自动量程功能并手动将数字式万用表的量程设置为000.0伏。
7.在信号电路端子1和低电平参考电压电路端子2之间安装一个数字式万用表。设置数字式万用表利用“MIN MAX（最小最大）”功能捕捉直流电压。
8.按下点火开关模式开关启动天线。
9.确认数字式万用表捕捉的最大电压高于1伏。
如果为1伏或更低
9.1 将点火开关置于“OFF（关闭）”位置，断开K84无钥匙进入控制模块处的线束连接器。
9.2 测试低电平参考电压电路和搭铁之间的电阻是否为无穷大。
如果电阻不为无穷大，则修理电路对搭铁短路故障。
如果电阻为无穷大
9.3 测试低电平参考电压电路端对端的电阻是否小于2欧。
如果为2欧或更大，则修理电路中的开路/电阻过大故障。
如果小于2欧，则更换K84无钥匙进入控制模块。
如果高于1伏
10.更换相应的T10无钥匙进入天线。
11.按下点火开关模式开关启动天线。
12.确认不显示“NO REMOTE DETECTED（未检测到遥控钥匙）”信息，且车辆起动。
如果显示“NO REMOTE DETECTED（未检测到遥控钥匙）”信息，且车辆不起动
更换K84无钥匙进入控制模块。
如果不显示“NO REMOTE DETECTED（未检测到遥控钥匙）”信息，且车辆起动
13.全部正常。
