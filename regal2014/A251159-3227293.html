6.2.1.1 音频放大器故障诊断仪信息
放大器故障诊断仪数据参数
参数
预期值
定义
运行条件： 将点火开关置于ON（打开）位置
Power Mode（电源模式）
Run（运行）
故障诊断仪显示Off（关闭）、Accessory（附件）、Run（运行）或Crank request（起动请求）。这是通过串行数据接收到的车辆电源模式状态。
Active Noise Cancellation（主动噪声消除）
On（打开）
故障诊断仪显示On（打开）或Off（关闭）。这表示主动噪声消除功能是否可供使用并能运行。
Engine Speed Signal（发动机转速信号）
Normal（正常）
故障诊断仪显示Normal（正常）或Malfunction（故障）。这是发动机转速信号专用电路的状态。
Battery Voltage（蓄电池电压）
Varies（变化）
故障诊断仪显示电压读数。这是至放大器输入端的电压。
Vehicle Speed（车速）
0公里/小时
故障诊断仪以公里/小时或英里/小时为单位显示放大器接收到的车速。
Engine Speed（发动机转速）
0转/分
故障诊断仪以转/分为单位显示放大器接收到的当前发动机转速。
Calculated System Temperature（计算的系统温度）
Varies（变化）
故障诊断仪以°C 或 °F为单位显示放大器的温度计算值。
Active Noise Cancellation Speaker Output（主动噪声消除扬声器输出）
OK（正常）
故障诊断仪显示OK（正常）或Malfunction（故障）。这是放大器扬声器输出端的状态。
Cylinder Deactivation System Command（气缸断缸系统指令）
Varies（变化）
故障诊断仪显示8 Cylinder Active（8个气缸启用）、Transitioning from 8 to 4（从8个气缸转变为4个气缸）、4 Cylinder Active（4个气缸启用）、Transitioning from 4 to 8（从4个气缸转变为8个气缸）或Malfunction（故障）。这是通过串行数据接收到的气缸断缸模式的状态。当未接收到此数据或发动机未运行时，则显示 Malfunction（故障） 。
Microphone Status（麦克风状态）
Normal（正常）
故障诊断仪显示Normal（正常）、Clipped（卡紧）或Malfunction（故障）。
Microphone 1 Input Level（麦克风1输入声级）
Varies（变化）
故障诊断仪显示从0到65,535的计数。这是麦克风1接收到的声级，数字越大表示声音越大。
Microphone 2 Input Level（麦克风2输入声级）
Varies（变化）
故障诊断仪显示从0到65,535的计数。这是麦克风2接收到的声级，数字越大表示声音越大。
Microphone 3 Input Level（麦克风3输入声级）
Varies（变化）
故障诊断仪显示从0到65,535的计数。这是麦克风3接收到的声级，数字越大表示声音越大。
Door Status（车门状态）
Varies（变化）
故障诊断仪显示All Closed（所有车门关闭）或One or More Open（一个或多个车门打开）。这是通过串行数据接收到的车辆进入系统车门锁闩的当前状态。
Folding Top Status（可折叠车顶状态）
Varies（变化）
故障诊断仪显示Closed（关闭）、Open（开启）、Malfunction（故障）或Normal（正常）。这是通过串行数据接收到的车辆可折叠车顶（如装备）的当前状态。
Calibration Part Number 1（校准零件号1）
Varies（变化）
故障诊断仪显示软件校准零件号1。
Calibration Part Number 2（校准零件号2）
Varies（变化）
故障诊断仪显示软件校准零件号2。
Calibration Part Number 3（校准零件号3）
Varies（变化）
故障诊断仪显示软件校准零件号3。
Calibration Part Number 4（校准零件号4）

故障诊断仪显示软件校准零件号4。

故障诊断仪显示软件校准零件号5。

故障诊断仪显示软件校准零件号7。
Calibration Part Number 8（校准零件号8）

Varies（变化）

Varies（变化）
