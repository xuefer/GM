电路/系统说明
媒体导向系统传输 (MOST) 信息娱乐系统网络是独立于GMLAN的专用高速多媒体数据流总线。媒体导向系统传输总线将被配置成物理硬线回路，总线内的每个装置都按设定的顺序在指定的媒体导向系统传输地址上发送和接收数据。媒体导向系统传输总线上的每个装置都需要有双绞铜线（2条TX发送线路，2条RX接收线路，和1条电子控制线路，即12伏唤醒信号线路）。A11收音机是媒体导向系统传输主控装置，并且将监测总线上的车辆配置、信息娱乐系统数据信息和错误。媒体导向系统传输初始化包括电子控制线路上的一个100毫秒低电压短脉冲，该线路连接至媒体导向系统传输环路所包含的所有装置。当MOST设备接收到此唤醒信息时，将会通过一个类属设备响应做出回应。一旦向A11收音机成功无误地报告了MOST总线上的这些初始响应，接下来的响应将报告MOST设备的地址及其功能要求和能力范围。此时，A11收音机将读入此信息并记录媒体导向系统传输总线上的地址节点次序。此节点地址列表现在将在A11收音机内被存储为媒体导向系统传输总线配置（在故障诊断仪数据显示屏幕上的名称为“Last Working MOST ID of Node 1-9（最近工作的媒体导向系统传输节点识别号1-9）”）。
当检测到MOST接收、发送或控制线路故障时，将不能根据唤醒请求按预期接收到发送/接收信息。然后，A11收音机和K74人机界面控制模块将执行诊断以隔离这些媒体导向系统传输故障。如果媒体导向系统传输控制线路对0伏低电压短路并持续过长的时间，A11收音机将设置U2098故障诊断码且K74人机界面控制模块将设置U0029 02故障诊断码。此时，媒体导向系统传输总线将无法进行通信，直到媒体导向系统传输控制线路短路被修复。
一旦通过媒体导向系统传输控制线路短路诊断，A11收音机就将尝试在媒体导向系统传输控制线路上重新发送初始短脉冲冲击多达3次。如果未收到预期的响应，A11收音机将继续保持设置了U0028故障诊断码的故障模式，并且在DTC U0028设为当前故障诊断码时不断发送300毫秒长脉冲，此脉冲将使最远的上游发送设备能够在此MOST故障/诊断模式中成为代理MOST主控设备。当A11收音机接收到此新媒体导向系统传输主控装置的识别号时，可以根据故障诊断仪数据参数“Surrogate MOST Master Node Upstream Position（代用媒体导向系统传输主控装置节点上游位置）”来识别代用媒体导向系统传输主控装置。应使用故障诊断仪和示意图，利用A11收音机数据显示屏幕中的“Last Working MOST ID of Node 1-9（最近工作的MOST节点识别号1-9）”参数确定MOST总线的配置和方向。当存在故障时，其将在A11收音机上显示新启用的“Surrogate MOST Master Node Upstream Position（代理MOST主控节点上游位置）”。这将有助于确定MOST设备/总线/控制模块的故障位置。此时，MOST设备和源自代理MOST主设备的电路、发送、接收或控制线路将是诊断的可疑区域。这些故障可能与任何媒体导向系统传输发送、接收或控制线路双绞铜线有关，或者可能是装置内部故障。
运行故障诊断码的条件
•  系统电压介于9 - 16伏之间。
•  点火开关置于“ON（打开）”位置。
•  收音机打开。
设置故障诊断码的条件
发送、接收、电子控制线路或内部设备发生了故障。
设置故障诊断码时采取的操作
部分或整个信息娱乐系统的音响和显示屏可能无法工作。
清除故障诊断码的条件
MOST成功完成初始化并且MOST总线锁止。
诊断帮助
•  利用A11收音机的故障诊断仪数据显示，根据“Last Working MOST ID of Node 1-9（最近工作的MOST节点识别号1-9）”参数确定MOST设备的位置识别号和MOST总线配置。
•  如果MOST总线有故障，则一个点火循环后，“Last Working MOST ID of Node 1–9”（最近工作的MOST节点识别号1-9）可能会丢失，并重置为“None（无）。在此情况下，使用来自示意图MOST总线配置并消除来自配置的不可用于特定车辆的选装设备。根据选装内容，各车辆可能具有不同的MOST总线配置。
•  利用A11收音机的故障诊断仪数据显示，通过诠释“Surrogate MOST Master Node Upstream Position（代理MOST主控节点上游位置）”参数，来确定代理MOST主设备以及MOST环路中断发生的位置。没有MOST总线故障时，该值为“None（无）”。
•  务必在诊断MOST总线故障前，先确定正确的MOST总线配置。下面是一个用“Surrogate MOST Master Node Upstream Position（代理MOST主控节点上游位置）”参数确定故障位置的例子。记住，每辆车的MOST总线配置可能有差异，不可能与以下配置一样。这个例子是用来帮助了解MOST总线诊断策略的：
– 例如，如果MOST总线配置为A11收音机>T3音频放大器>K74人机界面控制模块>S136信息娱乐控制触摸板>P16组合仪表并返回至A11收音机。A11收音机故障诊断仪数据显示屏将显示如下：
•  最近工作的MOST节点识别号1：收音机
•  节点2上次工作的MOST ID：音频放大器
•  最近工作的MOST节点识别号3：人机界面控制模块
•  最近工作的MOST节点识别号4：信息娱乐控制触摸板
•  最近工作的MOST节点识别号5：组合仪表
– 环中断诊断阶段过后，MOST总线主机A11收音机指示检测到环中断并设置DTC U0028。此时，A11收音机的故障诊断仪数据显示屏上会显示“Surrogate MOST Master Node Upstream Position”（代替MOST主机节点上游位置）参数。A11收音机分配“Surrogate MOST Master Node Upstream Position（代理MOST主节点上游位置）”值大约需要10秒。从MOST设备列表的底部起，向上计数以判定代替MOST主机设备。MOST环中断将从代替MOST主机上游发现。因为这将是暂停的节点并在MOST环中断期间进行MOST节点分配。
•  如果“Surrogate MOST Master Node Upstream Position”（代替MOST主机节点上游位置）数值为0，则代替MOST主机也是A11收音机。MOST环路中断在P16组合仪表，或者在A11收音机和P16组合仪表之间。
•  如果代替MOST主机节点上游位置数据为1，则代替MOST主机是P16组合仪表，它是来自A11收音机节点1的上游。MOST环中断在S136信息娱乐控制触摸板，或者在P16组合仪表和S136信息娱乐控制触摸板之间。
•  如果代替MOST主机节点上游位置数据为2，则代替MOST主机是S136信息娱乐控制触摸板，它是来自A11收音机节点2的上游。MOST环中断在K74人机界面控制模块，或者在S136信息娱乐控制触摸板和K74人机界面控制模块之间。
•  如果“Surrogate MOST Master Node Upstream Position”（代理媒体导向系统传输主节点上游位置）值为3，则代理媒体导向系统传输主控装置是K74人机界面控制模块，来自A11收音机节点3的上游。MOST环中断在T3音频放大器，或者在K74人机界面控制模块和T3音频放大器之间。
•  如果代替媒体定向系统传输主机节点上游位置数值为4，则代替媒体定向系统传输主机为T3音频放大器，它也是自A11收音机节点4的上游。媒体定向系统传输环中断位于T3音频放大器和A11收音机之间。
•  安装了最新软件后，DTC U0028保存为历史故障诊断码时，A11收音机将报告“Surrogate MOST Master Node Upstream Position（代理MOST主节点上游位置）”值。U0028故障诊断码状态和“Number of MOST Communication Breaks（MOST通信中断次数）”参数，必须和“Surrogate MOST Master Node Upstream Position（代理MOST主节点上游位置）”参数一起使用，才能成功完成诊断。这种方法用助于捕获间歇性故障条件下的代理信息。每次MOST总线状态从“正常运行（锁止状态）”转变为“关闭状态”，MOST通信中断次数计数器将增加计数，并从0-65535进行累计。MOST通信中断次数计数器增加10次后，DTC U0028将设置。
•  有一点很重要，那就是成功修理故障后，需清除DTC U0028，并将“Surrogate MOST Master Node Upstream Position（代理MOST主节点上游位置）”值重置为“None（无）”。通过断开A11收音机电源、断开蓄电池电缆，或者在50次点火电源断开循环后，“Surrogate MOST Master Node Upstream Position（代理MOST主节点上游位置）”值可重置为“None（无）”。这也将使MOST通信中断次数计数器重置为“0”。
•  MOST控制线路开路或者MOST传输/接收线路有故障将会导致设置DTC U0028。

修理检验
1.将点火开关置于“OFF（关闭）”位置，关闭所有检修孔盖，停用所有车辆系统，使所有钥匙距离车辆至少3米（9.8英尺）。所有车辆系统断电可能需要2分钟时间。
2.将点火开关置于“ON（打开）”位置。
