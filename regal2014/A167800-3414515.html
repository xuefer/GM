6.1.2.27 侧面物体传感器的编程和设置
注意:
•  切勿对侧面物体传感器 - 左进行编程，除非有维修程序或维修通讯的指导。如果侧面物体传感器 - 左未利用正确的校准软件进行正确配置，则侧面物体传感器 - 左无法正确控制所有车辆功能。
•  确保编程工具装备了最新软件并且与数据链路连接器牢固连接。如果编程过程中断，则可能会使编程失败或损坏侧面物体传感器 - 左。
•  在编程过程中，蓄电池电压稳定很重要。任何波动、峰值、过压或电压损失都将中断编程。需要时，安装EL-49642维修编程系统编程支持工具以维持系统电压。如果无法提供此工具，则连接一根电压充足的12伏跨接线或从交流电压电源上断开的助力器组件。不要连接蓄电池充电器。
•  关闭或停用可能对车辆蓄电池造成负荷的系统，如车内灯、车外灯（包括日间行车灯）、HVAC、收音机等。
•  在编程程序期间，遵循SPS提示以修正点火开关位置。
•  完成编程后，清除故障诊断码。清除动力传动系统故障诊断码可能会将Inspection/Maintenance (I/M)（检查/保养）系统状态显示设置为NO（否）。
诊断帮助
B218R侧面物体传感器模块 - 右不可单独编程。此模块由B218L侧面物体传感器模块 - 左（编程后）进行编程。如果更换右侧模块，则按照此程序更新模块。
更换和编程控制模块，或对其重新编程
若要编程一个更换的或已有的B218L侧面物体传感器模块 - 左，则执行以下程序：
1.进入维修编程系统 (SPS)，并按屏幕上的说明进行操作。
2.在SPS Supported Controllers（维修编程系统支持的控制器）屏幕上，选择B218L Side Object Sensor Module – Left - Programming（B218L侧面物体传感器模块 - 左 - 编程），并按屏幕上的说明进行操作。
3.在编程结束后，选择维修编程系统屏幕上的Clear All DTCs（清除所有故障诊断码）功能。
编程恢复失败
如果编程操作中断或失败，则执行以下步骤：
1.将点火开关置于ON（打开）位置。确保B218L侧面物体传感器模块 - 左、数据链路连接器和编程工具连接牢靠且维修编程系统软件是最新版本。
2.确认B218L侧面物体传感器模块 - 左不能重新编程。
如果B218L侧面物体传感器模块 - 左不能重新编程
2.1 将点火开关置于OFF（关闭）位置一分钟，然后将点火开关置于ON（打开）位置。
2.2 确认B218L侧面物体传感器模块 - 左不能重新编程。
如果B218L侧面物体传感器模块 - 左不能重新编程，则将其更换。
如果B218L侧面物体传感器模块 - 左能够重新编程。
2.3 全部正常。
如果B218L侧面物体传感器模块 - 左能够重新编程
3.全部正常。
维修指南
完成维修后执行诊断修理检验。
