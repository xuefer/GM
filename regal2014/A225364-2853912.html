6.1.2.20 媒体光盘播放器的编程和设置
注意:
•  切勿对控制模块进行编程，除非有维修程序或维修通讯的指导。如果控制模块未利用正确的校准软件进行正确配置，则控制模块无法正确控制所有车辆功能。
•  确保编程工具装备了最新软件并且与数据链路连接器牢固连接。如果编程过程中断，则可能会使编程失败或损坏控制模块。
•  在编程过程中，蓄电池电压稳定很重要。任何波动、峰值、过压或电压损失都将中断编程。需要时，安装EL-49642维修编程系统编程支持工具以维持系统电压。如果无法提供此工具，则连接一根电压充足的12伏跨接线或从交流电压电源上断开的助力器组件。不要连接蓄电池充电器。
•  关闭或停用可能对车辆蓄电池造成负荷的系统，如车内灯、车外灯（包括日间行车灯）、HVAC、收音机等。
•  在编程程序期间，遵循SPS提示以修正点火开关位置。
•  完成编程后，清除故障诊断码。清除动力传动系统故障诊断码可能会将Inspection/Maintenance (I/M)（检查/保养）系统状态显示设置为NO（否）。
诊断帮助
•  在编程期间，可能需要根据车辆装备选择多个校准。在以下程序中，车辆装备/常规选装件信息是否可用，以确保选择了正确的校准。
•  根据模块的当前软件等级或更新的发布，更换/编程控制模块时可能需要USB编程。如果应用的USB文件通过SPS无法获取，或尚未通过另一种分发方式从通用汽车公司收到，将不需要执行USB更新。
参考信息
专用工具
EL-49642 编程支持工具 (SPS)
关于当地同等工具，参见专用工具。
更换和编程控制模块，或对其重新编程
要对一个更换的或已有的控制模块进行编程，则执行以下程序：
不带后排座椅视频娱乐系统
控制模块不需要维修编程系统编程或任何设置程序。
带后排座椅视频娱乐系统
1.安装EL-49642维修编程系统编程支持工具。
2.将USB驱动器连接至计算机。
注意: 如果使用由通用汽车公司提供的已预先配置的USB驱动器，则转至步骤8。
4.进入维修编程系统 (SPS)，并按屏幕上的说明进行操作。
5.在维修编程系统支持的控制器屏幕上，选择Media Disc Player - USB File Transfer（媒体光盘播放器 - USB文件传输），并按屏幕上的说明进行操作。
注意:
•   如果没有可供USB通过维修编程系统传输的文件，则转至步骤11。
•   当复制新文件时，USB驱动器上所有原有文件将被删除。
6.文件传输完成后，马上将USB驱动器从计算机上移除。
7.点火开关处于ON（开启）位置，信息娱乐系统处于ON（开启）位置。
8.将USB驱动器连接至车辆的后排座椅USB接口。
9.信息娱乐系统将会识别更新文件是否可用。按照信息娱乐显示器上说明进行操作，当出现提示时选择“Update（更新）”。编程过程可能需要几分钟。
10.编程完成后，移除USB驱动器。按照信息娱乐显示器上说明进行操作。
11.在维修编程系统支持的控制器屏幕上，选择Media Disc Player - Programming（媒体光盘播放器 - 编程），并按屏幕上的说明进行操作。
12.在编程结束后，选择维修编程系统屏幕上的Clear All DTCs（清除所有故障诊断码）功能。
编程恢复失败
如果编程操作中断或失败，则执行以下步骤：
1.将点火开关置于ON（打开）位置。确保控制模块、数据链路连接器和编程工具连接牢靠且维修编程系统软件是最新版本。
2.确认控制模块能够重新编程。
如果控制模块无法重新编程
2.1 将点火开关置于OFF（关闭）位置一分钟，然后将点火开关置于ON（打开）位置。
2.2 确认控制模块能够重新编程。
如果控制模块无法重新编程，则更换控制模块。
如果控制模块能够重新编程。
2.3 全部正常。
如果控制模块能够重新编程
3.全部正常。
